# 雾下余粮（Tiny Signal V1.3.5）

一个轻量化的“搜打撤 + 资源管理”H5 游戏 Demo。

当前实现状态与需求分析基线见 [`docs/design/current-game-status.md`](./docs/design/current-game-status.md)。

玩家从庇护所进入可配置的雾蚀林缘，在战争迷雾中逐格探索。户外污染会持续消耗疯狂抗性；明雷接触后切换到独立回合制战斗，胜利后切割带独立污染值的异变肉，最后前往撤离点保住本次资源。回到庇护所后，可以将有限的圣遗物净化值用于恢复抗性或净化异变肉。

饥饿只在户外地图移动时消耗；战斗、切割、等待、撤离读条和使用道具不会额外扣除饥饿。

## 当前内容

- 主菜单：开始游戏 / 继续游戏 / 配置 / 关于 / 重置存档
- 庇护所：食物库存、食用、固定装备、灰麦种植格、可交互圣遗物
- 疯狂抗性：新游戏默认 10/10；户外默认每 5 秒产生 0.1 点环境污染，先扣抗性再将溢出加入疯狂
- 怪物肉净化：每块肉保存当前/最大疯狂；玩家可在圣遗物面板选择具体肉块进行完全或部分净化，食用只吸收当前剩余污染
- 户外：10×10～50×50 格子地图、跟随玩家的 20×20 视口、四方向移动、离散地图回合
- 战争迷雾：未探索 / 已探索记忆 / 当前可见三种状态
- 明雷敌人：普通敌人拥有四方向朝向和受障碍遮挡的可配置扇形视野；视野外继续真实行动，旧位置只显示记忆标记
- 危险反馈：Alert 警觉、Chase 追踪与 AttackIntent 攻击意图预告
- 独立战斗：宝可梦式转场、攻击/防御/道具/逃跑、速度决定先后手
- 怪物：统一地图状态机，支持 Idle / Wander / Alert / Chase / AttackIntent / Return / Cooldown / Dead
- 四套参数预设：静止型、游荡者、追踪者、可产出子怪的腐化巢穴
- 尸体切割：读条、肉量、背包容量与切割倍率
- 配置系统：分类表单、可视化地图编辑、Seed 随机布点、地图草稿试玩、本地保存、恢复默认、校验、JSON 导入/导出
- Demo 目标：3 次成功撤离并累计带回 12 块怪物肉
- 首次引导：目标、移动、迷雾、警觉、战斗、切割、异变肉、巢穴和撤离九个触发节点，可跳过并在配置页重置
- 探索结算：展示本次回合、格数、击杀、巢穴、异变肉、疯狂变化、剩余抗性、场景吸收的战斗伤害和明确失败原因
- 随机事件：8 个数据驱动事件，包含选择、条件与即时结果
- 疯狂表现：阶段文案、低语、边缘效果、脉动及减少动态效果
- 音频：22 个确定性生成的低保真 WAV 音效，支持缓存、冷却、变体、静音、音量和加载失败合成回退
- 视觉：弱化格线的潮湿地表、连续弧形敌人视野、贴身朝向指针与不规则柔化战争迷雾

## 操作

- 移动：方向键或 WASD
- 接敌：尝试进入敌人所在格；相邻但位于敌人背面不会仅因距离自动暴露
- 战斗：方向键切换行动，回车确认；“道具”中可以选择异变肉块
- 切割 / 撤离：站在目标格后按 E 或点击交互按钮
- 手机：使用屏幕方向键和操作按钮

异变肉可以恢复生命与户外饥饿，并按每块肉的当前污染提高疯狂；食肉污染不会被疯狂抗性抵消。疯狂阶段越高，攻击倍率越高。这是当前 Demo 的核心风险收益选择。

## Demo 胜利条件

在庇护所食物耗尽前完成 3 次成功撤离，并累计带回至少 12 块怪物肉。外出失败达到 3 次，或庇护所两类食物同时耗尽时，本轮 Demo 失败。

配置入口可以调整目标、玩家和敌人数值、疯狂阶段、地图事件、UI 表现和音量。配置保存后在下一次开始游戏时生效。

## 地图编辑、Seed 与巢穴

- 进入“配置 → 地图编辑”，可修改地图尺寸，放置障碍、出生点、撤离点、普通怪物或腐化巢穴，创建矩形随机怪物区域，并选中对象编辑属性；支持 50 步撤销/重做和单张地图 JSON 导入导出。“清空对象”清除障碍、固定怪物和随机区域，保留出生点与撤离点。
- 开启固定 Seed 后，相同地图配置和 Seed 会复现相同随机怪物点位；Seed 可以生成或复制。
- “试玩当前地图”直接运行尚未保存的草稿，退出后返回编辑器，不覆盖正式存档。
- 户外探索会保存 Seed、伪随机生成器状态、地图事件进度、环境污染间隔进度与运行时快照；刷新后不会重新随机已有点位、重置巢穴计时或改变后续运行时随机结果。环境污染从已保存进度继续，页面关闭或不可见期间不推进；战斗与事件选择界面不在快照中，刷新会回到最近一次已保存的户外探索状态。
- 腐化巢穴仍属于统一怪物配置，通过 `spawnConfig` 定时产出子怪；破坏后停止产出，已有子怪继续存在，并提供独立地图轮廓、首次发现、产出和破坏反馈。
- 巢穴不会主动接敌；玩家尝试进入巢穴所在格时，会主动进入普通战斗。
