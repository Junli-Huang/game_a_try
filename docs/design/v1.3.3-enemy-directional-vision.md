# V1.3.3 敌人朝向、扇形视野与观察行为

## 1. 版本目标

V1.3.3 在现有格子探索、敌人统一状态机和三态战争迷雾上增加方向性感知。普通敌人拥有可持久化的四方向朝向，只能发现处于视野距离、视野夹角且未被障碍遮挡的玩家。

本版不重写 AI 状态机，不增加声音感知、像素级圆锥、八方向美术或独立潜行数值。

## 2. 数据结构

普通敌人运行态增加：

```js
{
  facing: 'north' | 'east' | 'south' | 'west',
  lastSeenPlayerPosition: { x, y } | null,
  lastMove: { x, y } | undefined
}
```

统一怪物配置增加：

```js
vision: {
  enabled,
  range,
  angle,
  rotateWhenIdle,
  canRotateBeforeMove,
  canDetectAfterMove
}
```

`vision.range` 是当前方向性感知的有效距离；旧 `detectRange` / `detectRadius` 只保留用于兼容旧配置迁移。UI 的 `showEnemyVision` 只控制覆盖层，不关闭 AI 感知。

默认值：

| 预设 | 视野 |
| --- | --- |
| 静默菌兽 | 2 格、90°，低活动但可转头 |
| 游荡者 | 3 格、90° |
| 寻迹者 | 5 格、90° |
| 腐化巢穴 | 关闭 |

## 3. 扇形与遮挡算法

`getVisionCells()` 是不依赖渲染的纯函数：

1. 枚举以敌人为中心、`range` 为半径的地图内格子；
2. 使用格子中心的欧氏距离排除超距格；
3. 使用朝向单位向量和目标向量的点积计算夹角；
4. 夹角等于扇形边界时计入视野；
5. 使用 Bresenham 风格格线追踪检查障碍；
6. 障碍格本身可见，障碍之后的格子不可见。

玩家战争迷雾在启用 `terrainBlocksVision` 时复用同一个格线遮挡函数，避免形成两套障碍定义。默认玩家迷雾仍保持现有配置行为。

## 4. 敌人行动顺序

单次敌人行动：

```text
按当前状态调整朝向
→ 进行移动前视野判定
→ 新发现玩家则进入 Alert 并结束该敌人的本阶段行动
→ 否则执行原有等待、移动、追踪或返回逻辑
→ 移动成功后朝向移动方向
→ 配置允许时进行移动后视野判定
```

转向只是敌人行动阶段内的附属操作：

- 不增加地图回合；
- 不扣除移动能力或玩家饥饿；
- 不推进切割、撤离或环境污染；
- 不产生额外攻击；
- 同一阶段发现玩家后不立即追击或接敌。

## 5. 状态朝向

- `Idle` / 未接敌 `Wander`：每个敌人行动阶段从左转、保持、右转中选择；真正移动后以移动方向覆盖。
- `Alert`：朝向最后目击位置，继续执行现有警觉倒计时。
- `Chase`：朝向最后目击位置或现有追踪目标，移动后朝向路径下一格。
- `AttackIntent`：朝向追踪目标；仍须等待下一敌人阶段才进入战斗。
- `Return`：朝向 Home 路径下一格。
- `Cooldown`：存在最后目击位置时优先朝向该位置。
- `Dead`：不计算或显示视野。
- 腐化巢穴：保留兼容朝向字段，但视野关闭，不主动发现玩家。

玩家与敌人相邻但处于背面时不会仅因相邻自动触发 `Alert`。玩家主动尝试进入敌人或巢穴所在格，仍按既有规则主动接敌。

## 6. 地图表现与迷雾

- 只有敌人所在格当前对玩家可见时，才计算并绘制其实时视野覆盖。
- 覆盖层位于地形之上、单位和地形图标之下。
- 未探索区域不绘制覆盖；已探索的敌人记忆不显示或更新实时视野、朝向。
- 普通、Alert、Chase / AttackIntent、Cooldown 使用现有危险色系的不同透明度。
- 普通敌人图标右下角显示四方向小箭头；不增加美术资源。
- 死亡敌人、巢穴和关闭视野的敌人不显示覆盖与方向标记。

## 7. 存档和迁移

新快照直接保存 `facing`、`lastMove` 和 `lastSeenPlayerPosition`。旧快照恢复顺序：

1. 使用已有 `facing`；
2. 使用最近移动方向；
3. 使用由快照 Seed、敌人 ID 和位置生成的稳定方向。

相同旧快照反复加载会得到相同朝向，不使用每次加载变化的随机数。

旧怪物配置使用 `detectRadius` 或 `detectRange` 补入 `vision.range`，默认角度为 90°；旧巢穴视野默认关闭。

## 8. 验收

- 正前方、背后、边界、超距、障碍、四朝向和地图边缘计算正确；
- 转向后仍能在同一敌人阶段移动，且不额外结算世界资源；
- 方向性感知接入 `Alert → Chase → AttackIntent`，不跳过警觉和攻击意图；
- 可见敌人显示视野与箭头，迷雾记忆不泄露实时信息；
- 新旧快照与配置可恢复；
- 自动测试和生产构建通过。
