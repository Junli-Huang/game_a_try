## 正常流程

- [ ] 新游戏显示目标引导，可逐条确认或跳过全部。
- [ ] 外出后移动、迷雾、警觉、战斗、切割、异变肉和撤离引导在首次条件满足时出现。
- [ ] 关闭引导后输入恢复，刷新后已完成引导不重复。
- [ ] 战斗胜利后切割尸体，抵达撤离点并成功撤离。
- [ ] 撤离面板的回合、格数、击杀、巢穴、肉与疯狂变化正确。
- [ ] 返回庇护所、刷新页面并继续游戏，存档与目标进度正确。

## 失败与最终结算

- [ ] 分别验证战斗和饥饿失败原因、丢失肉量、失败次数与剩余机会。
- [ ] 达成目标后显示总统计、收束文案及继续/重开/主菜单操作。
- [ ] 达到失败上限和食物耗尽时分别显示明确原因。

## 巢穴流程

- [ ] 首次看见巢穴触发引导，巢穴按间隔产怪并遵守上限。
- [ ] 击杀子怪后可再次产怪；主动进入巢穴格触发战斗。
- [ ] 破坏巢穴后统计加一，且巢穴停止产怪。

## 地图编辑与隔离

- [ ] 修改尺寸、放置出生点/撤离点/怪物/巢穴并添加随机区域。
- [ ] 试玩未保存草稿后返回编辑器，正式配置和正常存档均未变化。

## 响应式布局

- [ ] 四个目标视口无横向溢出或主要区域遮挡。
- [ ] 屏幕方向键连续可用且不滚动页面；战斗二级菜单不越界。
- [ ] 切割/撤离按钮可见，结算可滚动，横竖屏切换不崩溃。
